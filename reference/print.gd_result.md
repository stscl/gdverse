# print GD result

S3 method to format output for GD model from
[`gd()`](https://stscl.github.io/gdverse/reference/gd.md).

## Usage

``` r
# S3 method for class 'gd_result'
print(x, ...)
```

## Arguments

- x:

  Return by [`gd()`](https://stscl.github.io/gdverse/reference/gd.md).

- ...:

  (optional) Other arguments passed to
  [`knitr::kable()`](https://rdrr.io/pkg/knitr/man/kable.html).

## Value

Formatted string output
