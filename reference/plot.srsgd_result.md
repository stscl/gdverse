# plot SRSGD result

S3 method to plot output for SRSGD model result in
[`srsgd()`](https://stscl.github.io/gdverse/reference/srsgd.md).

## Usage

``` r
# S3 method for class 'srsgd_result'
plot(x, ...)
```

## Arguments

- x:

  Return by
  [`srsgd()`](https://stscl.github.io/gdverse/reference/srsgd.md).

- ...:

  (optional) Other arguments.

## Value

A ggplot2 layer
