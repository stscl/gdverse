# plot interaction detector result

S3 method to plot output for interaction detector in
[`geodetector()`](https://stscl.github.io/gdverse/reference/geodetector.md).

## Usage

``` r
# S3 method for class 'interaction_detector'
plot(x, alpha = 1, ...)
```

## Arguments

- x:

  Return by
  [`geodetector()`](https://stscl.github.io/gdverse/reference/geodetector.md).

- alpha:

  (optional) Picture transparency. Default is `1`.

- ...:

  (optional) Other arguments passed to
  [`ggplot2::theme()`](https://ggplot2.tidyverse.org/reference/theme.html).

## Value

A ggplot2 layer
