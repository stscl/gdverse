# print spatial rough set-based ecological detector

S3 method to format output for spatial rough set-based ecological
detector in
[`srsgd()`](https://stscl.github.io/gdverse/reference/srsgd.md).

## Usage

``` r
# S3 method for class 'srs_ecological_detector'
print(x, ...)
```

## Arguments

- x:

  Return by
  [`srsgd()`](https://stscl.github.io/gdverse/reference/srsgd.md).

- ...:

  (optional) Other arguments passed to
  [`knitr::kable()`](https://rdrr.io/pkg/knitr/man/kable.html).

## Value

Formatted string output
