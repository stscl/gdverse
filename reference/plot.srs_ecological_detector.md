# plot spatial rough set-based ecological detector

S3 method to plot output for spatial rough set-based ecological detector
in [`srsgd()`](https://stscl.github.io/gdverse/reference/srsgd.md).

## Usage

``` r
# S3 method for class 'srs_ecological_detector'
plot(x, ...)
```

## Arguments

- x:

  Return by
  [`srsgd()`](https://stscl.github.io/gdverse/reference/srsgd.md).

- ...:

  (optional) Other arguments passed to
  [`ggplot2::theme()`](https://ggplot2.tidyverse.org/reference/theme.html).

## Value

A ggplot2 layer
