# print SPADE power of spatial and multilevel discretization determinant

S3 method to format output for SPADE power of spatial and multilevel
discretization determinant from
[`spade()`](https://stscl.github.io/gdverse/reference/spade.md).

## Usage

``` r
# S3 method for class 'spade_result'
print(x, ...)
```

## Arguments

- x:

  Return by
  [`spade()`](https://stscl.github.io/gdverse/reference/spade.md).

- ...:

  Other arguments.

## Value

Formatted string output
