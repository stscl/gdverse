# plot OPGD result

S3 method to plot output for OPGD model result in
[`opgd()`](https://stscl.github.io/gdverse/reference/opgd.md).

## Usage

``` r
# S3 method for class 'opgd_result'
plot(x, ...)
```

## Arguments

- x:

  Return by
  [`opgd()`](https://stscl.github.io/gdverse/reference/opgd.md).

- ...:

  (optional) Other arguments.

## Value

A ggplot2 layer
